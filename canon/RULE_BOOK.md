# Rule Book

> Immutable fair-play laws · compile stack root
> Governs all prose emission from Story Engine

---

## Golden Rule (First Law)

**Never lie to the reader.**

Every sentence on Read 1 must remain factually true on Read 5. Meaning may reframe; facts may not change.

---

## Compile Stack

Ordered gates before any prose reaches `manuscript/`:

```txt
1. Golden Rule audit        →  no factual deception
2. Time anchor check        →  fair-play clues present per movement
3. Recognition alignment    →  Lock N reframes, does not contradict
4. Visibility audit         →  important ≠ visible on Read 1
5. Adventure strip test     →  Read 1 satisfies without hidden layers
6. Originality firewall     →  no copied plots/characters/worlds
7. Book of Secrets gate     →  apex truths author-only; not leaked
8. Open fork check          →  unresolved items not smuggled as fact
```

## Recognition Language

| Use | Avoid |

---

## Vocabulary Ban (reader-facing)

Never reader-facing: "layers," "locks," "puzzle," "game," "clues," "hidden," "meta."

Never hint that readers are "supposed" to reread. Earned reread only (CF-CON-047).

No author biography, endorsements, or read-twice instructions in the published artifact (CF-CON-048).

Internal engineering vocabulary stays in `architecture/` and `canon/`.

---

## Chronology Rules

1. Published order ≠ temporal order (Chronology Lie)
2. Every movement contains at least one time anchor
3. Anchors must be fair on audit — present on Read 1, meaningful on reread
4. True timeline reconstructable from text alone

---

## Character Rules

1. No character exists solely as a twist delivery mechanism
2. Every POV character has dual chronology (beliefs vs truth) — details in Character Bible (Phase 6)
3. Antagonist force = Certainty, not a mustache-twirling villain

---

## Ending Rules

1. Invitation ending — reader completes meaning; answers *"Why did I think that happened?"* not *"What happened?"*
2. Final image must pass Sixth Sense audit (same facts, new meaning)
3. First/last sentence dyad locked before prose
