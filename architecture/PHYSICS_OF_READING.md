## Immutable law (CF-CON-089)

> **Typography is never decorative.**

If typography changes → **reality changed** — not author voice, not narrator emphasis.

```txt
TRUST GATE:  typography obeys universe even when it hurts the story
READER PAYOFF: "The typography never broke its own rules."
```

## Evidence not emphasis (CF-CON-090 · CF-CON-091)

Typography so **rare** many readers don't consciously remember seeing it.

| Function | Wrong | Right |
|---|---|---|
| Bold on Read 1 | Emphasis | Nothing noticed |
| Bold on Read 5 | Decoration recalled | **Evidence** assembled |

### Rarity benchmark (internal target)

```txt
100 pages · only SIX words ever bold (not six sentences)
Read 1:  nothing
Read 2:  interesting
Read 4:  six words form a sentence
Read 5:  that sentence was the answer
```

### Italic extreme (example pattern)

One word in entire novel italicized → Read 1: emphasis assumed → Recognition: only word narrator remembers correctly; all else reconstructed.

## Reality layers (CF-CON-082 refined · CF-CON-099)

The page is a **seismograph**. Tiny typographic shifts = tiny **fractures** (stress, not importance).

| Typography | Physics layer | Stress type |
|---|---|---|
| Roman / normal | **Observed reality** | Baseline measurement |
| *Italics* | **Believed reality** | Certainty · reconstruction |
| **Bold** | **Objective reality leaking through** | Collision · causal strain |
| Small caps | **Immutable truth** | Unchanging fact |

Font isn't changing. **Reality is.**

Reader never taught this. They **discover** it through Recognition.
