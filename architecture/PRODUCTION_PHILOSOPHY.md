# Production Philosophy

> Write the masterpiece · then compose the artifact
> CF-CON-117 · CF-ENTRY-031

---

## Film-editing model (CF-CON-111)

```txt
PHASE 1  →  Write the novel (story only)
PHASE 2  →  Compose the printed artifact (Composer's Pass)

Movie isn't shot in final timing — it's edited into rhythm.
Likewise: prose first · performance second.
```

**Not** writing to fixed layout during Phase 1. **Typesetting is the final movement of composition.**

## Two manuscripts (CF-CON-113)

```txt
NARRATIVE MANUSCRIPT          PERFORMANCE MANuscript
manuscript/narrative/         manuscript/performance/
────────────────────          ────────────────────────
Pure text                     Every page · margin · rest
No formatting decisions       Every typographic event
Story language                Every page turn · silence · visual beat
                              Edit EXPERIENCE not language
```

Phase 14+ authorization required before either exists as prose files.

---

## Composer's Pass (CF-CON-112)

Phase 2 after narrative lock:

- Every page · page turn · silence · paragraph break
- Every italic · every bold word (rare · physics-governed)
- **Orchestrating performance** — not editing prose

Example: page ends one word → turn → half page silence → one sentence. **Cannot exist in Word doc** — must be composed.

---

## Page turn (CF-CON-114)

One of the most powerful storytelling tools:

```txt
Reader cannot see beyond the turn = suspense · rhythm · silence
Compose page turns like Hitchcock composed edits
```

Scored in Performance Manuscript + [THE_SCORE.md](THE_SCORE.md).

---

## Author as typesetter (CF-CON-115)

Do **not** hand finished prose to external typesetter.

**Typography is part of the story.** Author becomes typesetter by necessity — not vanity.

---

## Original Score (CF-CON-116 · CF-CON-119 · CF-CON-120)

```txt
ONE canonical First Edition printing — the composed score
Every page · line · space fixed — like sheet music
Typography itself becomes collectible ("Original Score")

Later: paperbacks · translations · mass-market = PERFORMANCES
One score · many performances — not equal to the composed edition
```

Publishing note: legal/copyright metadata still required; reading experience design is canonical in First Edition only.

---

## Meaning accrues late (CF-CON-118)

Five-movement draft discipline — decisions deferred to appropriate layer:

| Movement | Focus | Layer |
|---|---|---|
| **1** | Story only | Narrative |
| **2** | Character · emotional truth | Narrative |
| **3** | Hidden clue architecture | Deep |
| **4** | Rhythm · cadence · silence | Deep + Rhythm Bible |
| **5** | Canonical performance edition | Surface · Performance MS |

Forces no typography/page-turn decisions in Movement 1. By Movement 5, clue architecture and rhythm are known — typographic signals land with maximum impact.

Aligns with [WATCHMAKER_PHASES.md](WATCHMAKER_PHASES.md) pre-writing pipeline **before** these production movements.

## Workflow map

```txt
WATCHMAKER PHASES 1–9     →  Story Engine artifacts (pre-prose)
MEANING ACCRUES LATE 1–4  →  Narrative Manuscript refinement
MOVEMENT 5                →  Performance Manuscript (Composer's Pass)
FIRST EDITION             →  Original Score (one canonical printing)
```

---

## Anti-patterns

```txt
⛔ Formatting decisions during Movement 1
⛔ External typesetter owns typography
⛔ Mass-market reflow treated as equal to Original Score
⛔ Page-turn choreography before clue architecture known
```
