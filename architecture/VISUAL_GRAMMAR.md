## Principle (CF-CON-081)

Typography is **not decoration**. It is an **invisible emotional score**.

```txt
FAIL:  reader notices typography as a system → pulled out of story
WIN:   reader feels cues unconsciously → Recognition reframes meaning
```

Fair play: **formatting never lies**. Reader assigned conventional meaning (emphasis, thoughts). Recognition reveals canonical meaning.

Fifth read realization: *"I've been reading typography as emphasis... but it was actually another language."*

---

## Limited vocabulary (CF-CON-084)

**One primary typeface only.** No multiple fonts (print · ebook · accessibility · translation constraints).

| Instrument | Canonical meaning | NOT |
|---|---|---|
| **Roman** | **Observed reality** | Default "neutral" |
| *Italics* | **Believed reality** | Thoughts · emphasis |
| **Bold** | **Objective reality leaking** | Emphasis · collision |
| Small caps | **Immutable truth** | Decorative caps |
| **Silence** | Rest · breath (not "white space") | Formatting |
| Indentation | Shift · entry into memory/register | Random layout |
| Em dash | Interruption · break in causal flow | Style choice |
| Ellipsis | Unfinished thought · trailing causality | Drama dots |

**That's the orchestra.** Each instrument = one meaning, used intentionally.

---

## Reality not speaker (CF-CON-083)

Typography identifies **how reality is behaving** — not **who** is speaking.

```txt
READ 1–3:  bold feels emotionally intense
READ 4:    bold marks causality under strain
TEXT:      unchanged
READER:    interpretation changed
```

Same Golden Rule as prose: facts invariant; meaning evolves.

---

## Voice via page breathing (CF-CON-085)

Distinct "voices" without font changes — **page breathes differently**:

| Voice rhythm | Visual feel |
|---|---|
| Long paragraphs · slow · flowing | Legato · sustained |
| Short · fragmented · percussive | Staccato · drum |

Reader senses difference **before** conscious recognition.

Pairs with [RHYTHM_BIBLE.md](RHYTHM_BIBLE.md) tempo/density scoring.

---

## Rarity principle (CF-CON-086)

> **If everything is accented, nothing is accented.**

Typography changes must be **incredibly rare**.

When they occur → **cymbal crash after quiet strings**.

Immunity to visual tricks = design failure.

---

## Absence as signal (CF-CON-082)

Deliberate **absence** of formatting is as meaningful as presence.

Unmarked roman in a field of certainty-italic passages = event.

---

## Recognition layer (CF-CON-087)

| Read | Reader assigns | Canonical truth (locked) |
|---|---|---|
| 1 | Bold = emphasis | Bold = collision |
| 1 | Italics = inner voice | Italics = certainty |
| 5 | Typography = decoration | Typography = second language |

Meaning never changes. Understanding does.

---

## Constraints as power (CF-CON-088)

Resist every available tool. **Restraint = elegance.**

One typeface · controlled emphasis · white space · rhythm · silence → every visual change becomes an **event**.

Novel feels immersive, not experimental.

---

## Compile gates (Visual Grammar)

Before any formatted text enters manuscript:

- [ ] Each typographic choice maps to exactly one canonical meaning
- [ ] No multiple typefaces
- [ ] Bold/italic usage passes rarity audit (cymbal-crash test)
- [ ] Typography marks reality state, not speaker ID
- [ ] Golden Rule: no deceptive formatting
- [ ] Accessibility fallback documented for ebook/translation
- [ ] Pairs with Rhythm Bible page score
