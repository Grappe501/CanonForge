## Discipline shift (CF-CON-069)

```txt
WRONG WORD:  write
RIGHT WORD:  compose

WRONG QUESTION:  "Is this sentence beautiful?"
RIGHT QUESTION:  "Does this note belong here?"
```

The medium is **time**, not paragraphs. Author discipline: percussionist thinking in **tempo**.

## White space as rest (CF-CON-071)

White space is **not formatting** — it is musical rest.

| Rest type | Percussion analog | Use |
|---|---|---|
| Single blank line | Quarter rest | Scene breath |
| Double blank line | Half rest | Memory gap |
| Page break | Whole rest | Movement shift |
| Extended gap | Fermata | Recognition pause |

### White space meaning shifts by read (Recognition layers)

| Read | White space means |
|---|---|
| 1 | Distance |
| 2 | Memory |
| 3 | Missing time |
| 5 | Invisible event marker (discovered, never told) |

## Emotional meter map (CF-CON-072)

**Internal only** — never labeled for reader.

| Emotional state | Meter | Feel |
|---|---|---|
| Wonder | 3/4 | Lift · swing |
| Conflict | 5/4 | Unease · displacement |
| Acceptance | 4/4 | Ground · resolve |
| Memory | Free time | Unmeasured · drift |
| Recognition | Syncopation | Expectation violated · reframe |

Composer knows the meter. Reader **feels** it.

## Percussion principles (CF-CON-078)

Good drummers don't keep time — they create **expectation**.

```txt
Expectation → Release → Expectation → Delay → Impact
```

Anti-skimming: not complexity — **rhythm**. Compelling cadence forces reader to slow and match tempo (poetry effect).

Neuroscience gate: readers skim when prediction succeeds; they slow when prediction **fails just enough** (not confusion — rhythmic displacement, delayed resolution).

## Composed prose example (template)

```txt
[Fast] [Fast] [Fast]
[Pause — one sentence]
[White space — quarter rest]
[Single word]
[White space]
[Avalanche — density spike]
```

Reader breathing changes. They are **performing**, not scanning.

## The heartbeat (CF-CON-073)

Recurring pulse throughout the score:

```txt
Three short sentences.
Pause.
Three short sentences.
Pause.
```

Read 1–3: perceived as style.
Read 4+: Recognition — pulse **measures something** (TBD in Book of Secrets / Clue Ledger).

## Breathing as instrument (CF-CON-074)

Compose for the **inner voice**, not the eye:

- Long flowing paragraphs → reader breathes slower
- Three one-word paragraphs → reader inhales sharply
- Story occupies **nervous system**, not just cognition

Target instruments: breathing · pulse · subvocalization · internal narration.

---

## Emotion via violated expectation (CF-CON-075)

Emotion = **expectation violated**, not adjective density.

```txt
A parent reaches for a child's hand.
Missing.
```

One sentence. No adjectives. Maximum impact.

---

## Silence as content (CF-CON-079)

> Silence isn't the absence of music. It's part of the music.

The **unwritten** parts may become as important as the written parts. Score must document intentional silences, not just words.

---

## Reread as rehearing

First reread: rhythm suddenly makes sense — reader anticipates cadence like hearing a song twice.

Each Recognition pass **changes the music** (CF-CON-077) without changing the text.
