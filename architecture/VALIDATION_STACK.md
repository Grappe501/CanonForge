# Validation Stack

> North stars · success metrics · compile gates

---

## North stars

1. **Story wins** — best adventure story framing; cleverness invisible
2. **Fair play** — Golden Rule never violated
3. **Reader changes** — book invariant; reader evolves
4. **Experience product** — the journey is the deliverable
5. **Organic evangelism** — readers discover genius themselves

---

## Success metrics

| Metric | Target | Source |
|---|---|---|
| Read 1 satisfaction | Complete emotional arc without hidden layers | CF-CON-028 |
| Organic reread | 7/10 finishers reread without prompting | CF-CON-029 |
| Hook rate | End forces page-one flip | CF-CON-027 |
| Discussion | Theory/timeline/clue conversations | CF-CON-027 |
| Fair-play audit | Zero factual lies across 5 Recognitions | CF-CON-023 |
| Adventure strip | Read 1 works with all systems removed | CF-CON-028 |

---

## Three-level success model

| Level | Gate |
|---|---|
| **L1 — First read** | Never-rereads reader still wins |
| **L2 — Hook** | "Timeline wasn't what I thought — I have to reread" |
| **L3 — Discussion** | Theory, timeline, clue conversations emerge organically |

---

## Compile gates (pre-prose)

Before any movement prose:

- [ ] Golden Rule audit pass
- [ ] Time anchor per movement mapped
- [ ] Lock 1–5 Recognition payloads defined
- [ ] Visibility vs importance audit
- [ ] Originality firewall check
- [ ] Open forks explicitly excluded from prose
- [ ] Book of Secrets compile gate (author sign-off)

---

## Anti-patterns

```txt
⛔ Puzzle first · novel second
⛔ Reader feels tricked
⛔ Meta vocabulary in prose
⛔ Infinite mystery sprawl (LOST failure mode)
⛔ Typographic gimmick dependency (House of Leaves failure mode)
⛔ Smuggling open forks as fact
```
