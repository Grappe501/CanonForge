# Story Engine Spec

> Container · compile workflow · pre-prose operating system

---

## Definition

The Story Engine is the **complete pre-prose artifact set** required before a single word of manuscript is written. It is not the book — it is the machine that produces the book.

```txt
INPUT:   conversation uploads · author decisions · Book of Secrets
PROCESS: watchmaker phases · validation stack · compile gates
OUTPUT:  manuscript/ (Phase 14+ ONLY)
```

## Compile workflow

```txt
1. INGEST    →  knowledge/entries/ append upload
2. REGISTER  →  CONCEPT_REGISTRY + concepts.json
3. DECIDE    →  DECISION_LOG (lock or mark open)
4. SYNTHESIZE →  MASTER_CANON bump
5. VALIDATE  →  VALIDATION_STACK gates
6. PHASE     →  advance watchmaker phase
7. EMIT      →  prose ONLY when Phase 14 authorizes
```

## Recursive novel contract

- Read 1: complete adventure (Pillar 1 test)
- Read 2–5: progressive Recognition via Five Locks
- Text invariant; reader interpretation evolves
- Every page audited against Golden Rule before inclusion
